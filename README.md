# healthcare-analytics
Healthcare Encounter Analytics: SQL Cleaning, EDA, Modeling, and Dashboarding
The dataset supports descriptive healthcare analytics better than predictive modeling because the available fields weakly, if at all, predict billing and test results.

## Assignment Requirements Covered

| Requirement | How This Project Addresses It |
|---|---|
| Healthcare dataset | Uses a public healthcare encounter dataset containing demographics, admissions, billing, conditions, medications, and test results. |
| SQL | Imports raw CSV data into MySQL, stages the data, removes duplicates, cleans fields, creates derived columns, and validates row counts. |
| Python | Performs exploratory analysis, summary statistics, visualization, and baseline predictive modeling. |
| Power BI | Visualizes billing, admissions, length of stay, test results, and condition-level trends in an interactive dashboard. |
| Real-world impact | Interprets how descriptive analytics can support resource planning, billing review, and operational reporting. |
